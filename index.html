Por que não torcer para o Corinthians?

Torcer para o Corinthians não é para qualquer um. Se você busca uma vida pacata, previsível e sem fortes emoções, o Alvinegro Paulista definitivamente não é para você. Não torcer para o Corinthians é a escolha de quem prefere poupar o coração.

Ser corinthiano significa abraçar o sofrimento, viver no limite do teste cardíaco e saber que nenhum jogo está ganho até o último apito do juiz. É uma torcida que não apoia porque o time está ganhando; apoia para o time ganhar. Se você não tem estômago para a "Fiel", para a mística da Neo Química Arena e para uma paixão que beira a loucura, é melhor ficar bem longe.

"O Corinthians não é só um time de futebol, é uma religião, uma filosofia de vida onde a lógica quase nunca faz sentido."
